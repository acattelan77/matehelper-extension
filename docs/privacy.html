MateHelper Privacy Policy

MateHelper is an independent, unofficial helper extension. It is not affiliated with, endorsed by, sponsored by, or officially connected to any third-party platform, customer, or vendor.

MateHelper does not collect, store, sell, share, or transmit user data.

The extension runs locally in Chrome on the supported site patterns declared in the extension manifest. It reads visible page text only to find plain-text http:// and https:// URLs and convert them into clickable links.

MateHelper does not use analytics, tracking, cookies, external servers, remote code, background network requests, or persistent storage.

No page content, browsing history, credentials, personal information, or translation content leaves the user's browser.

Chrome Web Store Privacy Summary

  • Single purpose: Make plain-text URLs clickable on supported web pages.
  • Data collection: No user data is collected.
  • Data sharing: No user data is sold, shared, transferred, or sent to third parties.
  • Local processing: Page text is processed locally in the browser only to identify URL-like text.
  • Permissions: The extension is limited to the supported site patterns declared in the extension manifest and requests no additional Chrome extension permissions.
